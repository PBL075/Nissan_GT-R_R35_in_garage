Nissan_GT-R_R35_in_garage
Acesta este un sistem digital de gestiune a garajului creat în C#, dedicat legendarului model Nissan GT-R R35.

Ce face programul:
Gestiune Date: Definește profilul tehnic complet al mașinii (Model, Cai Putere, Accelerație, Preț și Culoare).

Interfață Consolă: Afișează un raport detaliat și stilizat al mașinii aflate în garaj.

Verificare Status: Confirmă starea mașinii printr-un mesaj final: "Masina este gata de drum".

Detalii tehnice (Variabile):
string: Folosit pentru numele modelului ("Nissan GT-R R35") și culoarea ("Neagra").

int: Folosit pentru valori întregi precum puterea (570 HP) și prețul (120,000 EUR).

double: Folosit pentru precizie la timpul de accelerație de la 0 la 100 km/h (2.8 sec).
